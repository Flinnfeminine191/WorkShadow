## 📔 Key features

### Rich text notes
Write your daily logs using headers, bullet points, and lists. The built-in editor supports formatting options to help you organize thoughts. You can bold, italicize, or highlight text to make information easy to read.

### Data privacy
WorkShadow runs locally. It stores your journal files in your Documents folder. No cloud servers host your entries. You control your information at all times.

### Semantic search
Find your notes without remembering exact keywords. The search tool understands the meaning behind your text. If you search for "marketing meeting," it finds notes about "ads discussion" or "campaign planning." 

### AI summaries and Q&A
Connect your own AI model keys to the app. WorkShadow acts as a bridge between your local notes and modern intelligence. You can ask the app, "What did I finish last Thursday?" or "Give me a summary of this project." The app reads your local files and provides answers based on your history.

## 🛠 Setting up your own AI

The app supports multiple models. You can add your API keys to the settings menu.

1. Open the settings icon in the top right corner.
2. Select the AI tab.
3. Choose your provider from the dropdown menu.
4. Paste your API key into the input field.
5. Save your changes.

The app uses these keys only to send your questions to the model. It does not store your keys in the cloud. It hides them on your local drive for access only by the app.

## ❓ Frequently asked questions

### Does the app work without internet?
Yes. You only need the internet to perform the initial download and if you use cloud-based AI models. If you use a local model, you can run the app entirely offline.

### Can I change where my data stays?
Yes. Go to the settings menu and select the storage tab. Choose a new folder for your journal files.

### Is my content safe?
Your data belongs to you. Since it stays on your hard drive, it is as safe as your computer is. We advise you to back up your Documents folder regularly to prevent loss.

### Does the app track my usage?
No. The code contains no trackers. We do not receive logs or analytics about how you use the app.

## 🚀 Troubleshooting

If you have issues, check these steps:

- Check if you have the latest Windows updates.
- Ensure your internet connection is stable if you rely on internet-based AI.
- Restart the application if it becomes sluggish.
- Check the settings to ensure your API keys remain valid.
- Reinstall the application if you encounter critical errors.
